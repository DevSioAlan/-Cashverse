# Cashverse-Simulator-Backend
[🌌] Cashverse Simulator
Extraits du code serveur (Backend) d'un jeu type Simulator sur Roblox (Luau). Ce dépôt met en avant mes systèmes de monétisation, d'algorithmique (RNG pondéré pour l'éclosion d'œufs), de gestion de base de données (DataStore) et d'Anti-Cheat personnalisé côté serveur.
