# [🌌] Cashverse 💸

🌟 **Bienvenue dans Cashverse !** 🌟
Ton objectif est simple : **Devenir le joueur le plus riche de l'univers !** 💰

## 🎮 COMMENT JOUER

| Action | Description |
|--------|-------------|
| 💸 **Récolter** | Ramasse l'argent au sol pour construire ta fortune |
| 🐾 **Pets** | Fais éclore des Pets puissants pour t'aider à gagner plus vite |
| ⚡ **Rebirth** | Rebirth pour multiplier tes gains et devenir inarrêtable |
| 🌍 **Zones** | Débloque de nouvelles zones et découvre des trésors cachés |
| 🏆 **Classement** | Grimpe dans le classement et montre qui est le patron ! |

## 🎁 BONUS

- 👍 Lâche un **Like** et mets le jeu en **Favoris** ⭐ pour de nouvelles mises à jour !
- 👥 Rejoins le **Serveur Discord** pour obtenir des récompenses exclusives et des Pets gratuits !

> ⚠️ **Un bug ?!** Signale-le nous sur le Serveur pour qu'on le corrige au plus vite.

## 🗂️ Structure du projet

```
src/
├── ReplicatedStorage/
│   └── Modules/
│       ├── GameConfig.lua      # Configuration globale du jeu
│       ├── PetData.lua         # Données des Pets (rarité, multiplicateurs)
│       └── ZoneData.lua        # Données des Zones (coût, multiplicateurs)
├── ServerScriptService/
│   ├── GameManager.server.lua      # Gestion globale & sauvegarde
│   ├── MoneySystem.server.lua      # Système de collecte d'argent
│   ├── PetSystem.server.lua        # Système d'éclosion et de Pets
│   ├── RebirthSystem.server.lua    # Système de Rebirth
│   ├── ZoneSystem.server.lua       # Système de Zones
│   └── LeaderboardSystem.server.lua# Classement global
├── StarterPlayerScripts/
│   └── GameClient.client.lua       # Interface cliente & interactions
└── StarterGui/
    └── ScreenGui/
        └── MainUI.lua              # Interface utilisateur principale
```

## ⚙️ Installation (Rojo)

1. Installe [Rojo](https://rojo.space/) et [Roblox Studio](https://www.roblox.com/create).
2. Clone ce dépôt.
3. Lance `rojo serve` dans le dossier racine.
4. Connecte Rojo depuis Roblox Studio.
